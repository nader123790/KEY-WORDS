Lec 4 — Image Processing · Filters · Convolution · Noise
🖼️ Image Fundamentals
🔢
Image as Function
MATH
array / matrix of numbers (pixels) d=1 → grayscale d=3 → color (RGB) range: 0–255 (8-bit)
📐
Digital Image (Discrete)
MATH
sampled on regular grid finite pixel values not continuous integer values
🔧 Filters & Operations
📦
Box Filter (Mean)
LINEAR FILTER
average of neighborhood smoothing effect removes sharp features equal weights
🔔
Gaussian Filter
LINEAR FILTER
center pixels weigh most σ controls smoothing extent separable (rows then columns) natural blur large σ → heavy blur
🔃
Median Filter
NON-LINEAR FILTER
sort neighborhood → pick middle no new pixel values introduced robust to outliers / spikes best for salt & pepper noise
🔪
Sharpening Filter
LINEAR FILTER
accentuates differences with local average original − blurred + original enhances edges
⚫⚪
Thresholding
SEGMENTATION
intensity > T → white (1) else → black (0) binary image output separates objects from background
1️⃣
Unit Impulse (Identity)
SPECIAL FILTER
zeros everywhere, 1 at center output = input unchanged a * e = a
✖️ Convolution vs Cross-Correlation
✖️
Convolution ( * )
OPERATION
flip kernel (both dims) then dot product G = H * F reversed + shifted
🔍
Cross-Correlation ( ⊗ )
OPERATION
NO flip → direct dot product measure of similarity template matching / pattern detection max when signals match
📐 Convolution Properties
↔️
Properties of Convolution
MATH
Commutative: a*b = b*a Associative: a*(b*c) = (a*b)*c Scalars factor out: k(a*b) = a*(kb) Separable: 2D → two 1D passes Identity: a*e = a
